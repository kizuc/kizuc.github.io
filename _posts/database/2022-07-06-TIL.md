## SQL Plus & SQL Developer
쿼리 구문(DB관리 명령어-DBA, Object 관리 명령어-개발자)의 입력/실행/결과를 확인하는 프로그램

### SQL Plus
- 터미널 환경의 툴로 데이터베이스 관리자(DBA)가 주로 사용한다. 무거운 명령어를 사용하는 데 SQL Developer에서 실행하면 그래픽 요소를 거쳐 늦기 때문이다.

로컬 접속의 경우
```sql
conn hr/hr
```
원격 접속의 경우
```sql
conn hr/hr@IP주소
```

### SQL Developer
- 그래픽 환경의 툴로 개발자가 주로 사용한다.
